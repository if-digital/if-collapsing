Easy to build Tabs or Accordions by using this Gutenberg Block. No Javascript, only pure CSS.

== Description ==

This Plugin will add the IF.COLLAPSING block to your WordPress site. With the block you can create accordions or tabs nativ in the gutenberg editor.
The collapsing content is not restricted, so you can add all available blocks to it.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/if-collapsing` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= Do this plugin need jQuery =

No. The IF.COLLAPSING is plain css driven. No jQuery or other libraries are needed.
